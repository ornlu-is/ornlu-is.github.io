    Project Minus #

    What’s this?

    Project Minus is my personal blog, where I store all my notes on different topics (usually related to Statistics). All code is adequately linked to my Github and all content is properly referenced. I’ll also post some data analyses that I perform out of curiosity.

    About me

    I hold a BSc in Engineering Physics and a MSc in Applied Mathematics from Instituto Superior Técnico (a faculty of University of Lisbon). I currently work in Finance, but retain my curiosity and desire to learn more about Statistics and Machine Learning.

    A note on the content #

    Please keep in mind that all the content in this website is not proof-read by external sources, which means that there might be mistakes. Do not hesitate in letting me know if you find any mistakes.
